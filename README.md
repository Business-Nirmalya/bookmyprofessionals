# BookMyProfessionals.com

A website for booking professional services like electricians, plumbers, makeup artists, event planners, interior designers, architects, puja services, and more.

## Features

- Responsive design that works on all devices
- Professional categories with easy filtering
- Professional listing with ratings and prices
- Booking system with date/time selection
- User registration and login (demo)
- Testimonials section
- How it works guide

## Setup Instructions

1. Create a new GitHub repository
2. Upload all files (index.html, style.css, script.js, README.md)
3. Connect your domain (BookMyProfessionals.com) to GitHub Pages
4. Set up Cloudflare for DNS management and CDN

## GitHub Pages Deployment

1. Go to your repository settings
2. Navigate to "Pages" section
3. Set source to "main" branch
4. Your site will be published at `https://username.github.io/repository-name`
5. For custom domain, add `BookMyProfessionals.com` in the custom domain field

## Cloudflare Setup

1. Sign up for a Cloudflare account
2. Add your domain (BookMyProfessionals.com)
3. Update nameservers as instructed by Cloudflare
4. Configure DNS settings:
   - Add an A record pointing to GitHub Pages IP addresses
   - Enable SSL/TLS encryption
   - Configure caching and security settings

## Customization

- Update colors in CSS `:root` variables
- Replace placeholder images with your own
- Add more professionals to the JavaScript array
- Connect to a backend for real user authentication and booking management

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Font Awesome icons
- Google Fonts
- Unsplash for placeholder images
